# Data directory

FightSafe AI keeps **small, redistributable** annotation files in Git. Large media and experiment outputs are downloaded or generated locally.

## Included in the repository

| Path | Description |
|------|-------------|
| `tapko/annotations/jedi_submissions.json` | Draft TapKO reference windows (not visually confirmed ground truth) |
| `tapko/examples/` | Minimal TapKO annotation schema examples |
| `FightSafeBench/annotations/` | Sample student annotation spreadsheets |
| `FightSafeBench/events.csv` | Merged benchmark events (pilot subset) |
| `FightSafeBench/dataset_statistics.json` | Aggregated pilot statistics |
| `boxingvi/annotations/` | BoxingVI punch-interval spreadsheets (V1–V10) |
| `boxingvi/metadata/` | BoxingVI metadata |
| `annotations/case_*.json` | Manual interval labels for fusion case-study clips A–F |

## Not included (download or generate locally)

| Path | Size (approx.) | How to obtain |
|------|----------------|---------------|
| `tapko/videos/jedi_submissions.mp4` | ~200 MB | Download the instructional BJJ clip referenced in `jedi_submissions.json` (`video_url` field) with `yt-dlp` or place your own licensed copy at this path |
| `boxingvi/skeleton/` | ~50 MB | Obtain BoxingVI skeleton keypoints from the dataset authors ([Kumar et al., arXiv:2502.x](https://arxiv.org/abs/2502.00000)) and extract under `data/boxingvi/skeleton/` |
| `clips/` | varies | User-provided combat-sports clips for case studies |
| `runs/`, `outputs/` | varies | Generated by `fightsafe run-pipeline`, `risk-ablation-all`, TapKO detect/evaluate |

### Example: download TapKO pilot video

```bash
mkdir -p data/tapko/videos
# Replace URL with the value in data/tapko/annotations/jedi_submissions.json if needed
yt-dlp -f 'bv*+ba/b' -o 'data/tapko/videos/jedi_submissions.%(ext)s' '<VIDEO_URL>'
ffmpeg -i data/tapko/videos/jedi_submissions.webm -c copy data/tapko/videos/jedi_submissions.mp4
```

## Redistribution

- **BoxingVI** and **YouTube-sourced** clips may carry license restrictions. Do not redistribute downloaded media in public archives without rights review.
- Annotation JSON/CSV files in this repository are research drafts unless marked `visually_confirmed` in the TapKO schema.
- For Zenodo deposits, publish **software + annotation schemas**; link large video data via external DOIs or access-controlled storage.
